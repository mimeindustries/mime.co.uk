What is it?

The MeArm Maker is an easy-to-build robot arm kit that's designed to get children (and adults!) learning about technology, engineering and programming. It's been expressly designed to be easy to build (age 11+) and use.

It comes without the controller, leaving you to supply your own. It's easy to get it working with Arduino, Raspberry Pi, mBed and any other microcontroller platform you can think of (so long as it can control a servo!).

Note: animation shows an older version of the MeArm Pi

How do I build it?

The MeArm Maker has been designed from the ground up to be easy to assemble. Children can build it themselves and we suspect adults may be able to build it unaided too. The only tool you'll need to build it is a hex key, and we include one in the box!
